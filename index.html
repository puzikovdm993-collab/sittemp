Вкладка "Файл"

Управление файлами проекта:

  • Открыть - загрузка изображений с компьютера или сервера
  • Сохранить - сохранение результатов работы локально или на сервере
  • Открытые файлы - переключение между несколькими открытыми изображениями

Вкладка "Правка"

Работа с историей действий:

  • Отменить (Ctrl+Z) - отмена последнего действия
  • Повторить (Ctrl+Y) - повтор ранее отменённого действия
  • История (Ctrl+H) - просмотр и навигация по истории изменений

Вкладка "Вид"

Управление отображением изображения:

  • Увеличить - приближение изображения
  • Уменьшить - отдаление изображения
  • 100% - сброс масштаба к исходному размеру
  • По размеру окна - автоматическое масштабирование под размер экрана

Вкладка "Изображение"

Геометрические преобразования:

  • Обрезка - обрезка изображения по выделенной области
  • Поворот - поворот на 90°, 180°, 270°
  • Отражение - зеркальное отражение по горизонтали или вертикали

Вкладка "Фильтры"

Применение фильтров и обработка изображений:

  • Выделение контуров - фильтры для обнаружения границ и контуров
  • Поиск объектов - поиск окружностей, осей симметрии, центров
  • Предварительная обработка - аппроксимация, арифметические операции, удаление дефектов
  • Прочее - медианный фильтр, нормализация

Вкладка "Регулировка"

Настройка параметров изображения:

  • Основные настройки - поворот/масштабирование, функции окна, яркость/контраст
  • Цветокоррекция - баланс цвета, тон/насыщенность, уровни, кривые

Вкладка "Инструменты"

Специализированные инструменты:

  • Code Lab - выполнение пользовательских скриптов
  • СРТИ - исправление и фильтрация СРТИ
  • Фотохронометрия - фильтр ФХМ

Вкладка "Режимы"

Переключение режимов работы приложения:

  • Режимы - переход к workflow (сценариям обработки)

Вкладка "Настройки"

Параметры приложения:

  • Интерфейс - отображение цветовой шкалы, выбор палитры
  • Тема оформления - переключение светлой/тёмной темы

Панель инструментов

Инструменты для работы с изображением:

  • Курсор - выбор и перемещение по изображению, получение информации о пикселях
  • Профиль - построение графика интенсивности вдоль линии
  • Лассо - выделение произвольной области
  • Перемещение - перемещение выделенной области

Горячие клавиши

  • Ctrl+Z - Отменить действие
  • Ctrl+Y - Повторить действие
  • Ctrl+H - Открыть историю
  • Ctrl+S - Сохранить
  • Ctrl+O - Открыть файл
